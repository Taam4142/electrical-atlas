## 1. แนวคิดที่ใช้ได้ทันที

**อนุกรมและขนาน** คือสองวิธีพื้นฐานในการต่อชิ้นส่วนในวงจร

มันกำหนดว่าอะไรใช้ร่วมกัน:

- ในวงจร **อนุกรม** ชิ้นส่วนต่อเรียงกันทีละตัว ดังนั้นกระแสเดียวกันไหลผ่านทุกชิ้นส่วน
- ในวงจร **ขนาน** ชิ้นส่วนต่อคร่อม node คู่เดียวกัน ดังนั้นแรงดันเท่ากันปรากฏคร่อมทุกแขนง

แนวคิดเดียวนี้ปลดล็อกพฤติกรรมวงจรจำนวนมาก

<aside className="callout">

**ภาพจำแรก:** อนุกรมคือถนนเส้นเดียว ส่วนขนานคือถนนหลายเส้นระหว่างจุดสองจุดเดียวกัน

</aside>

## 2. Node คืออะไร?

**Node** คือจุด หรือกลุ่มจุดที่ต่อถึงกัน และมองได้ว่ามีแรงดันเดียวกัน

ถ้าขั้วของชิ้นส่วนสองตัวต่ออยู่ระหว่าง node คู่เดียวกัน ชิ้นส่วนนั้นอยู่แบบขนาน

ถ้ากระแสต้องผ่านชิ้นส่วนหนึ่งก่อน แล้วจึงผ่านอีกชิ้นส่วน โดยไม่มีทางแยกระหว่างกลาง ชิ้นส่วนนั้นอยู่แบบอนุกรม

รูปร่างของรูปวงจรอาจหลอกมือใหม่ได้ สิ่งที่สำคัญกว่าคือ node ต่อถึงกันอย่างไร

</section>

<section>

## 3. การต่ออนุกรม

ในทางอนุกรม มีเส้นทางให้ประจุไหลเพียงเส้นเดียว

ดังนั้น:

- กระแสเดียวกันไหลผ่านทุกชิ้นส่วน
- แรงดันจากแหล่งจ่ายถูกแบ่งคร่อมแต่ละชิ้นส่วน
- ความต้านทานรวมคือการบวกกัน
- ถ้าชิ้นส่วนหนึ่งเปิดวงจร เส้นทางทั้งหมดหยุดนำกระแส

สำหรับตัวต้านทาน:

`R_total = R1 + R2 + R3 + ...`

ตัวอย่าง:

`100 Ω + 220 Ω = 320 Ω`

ถ้าใช้แหล่งจ่าย `12 V`:

`I = V / R = 12 / 320 = 0.0375 A`

ดังนั้นกระแส `37.5 mA` เท่ากันไหลผ่านตัวต้านทานทั้งสองตัว

</section>

<section>

## 4. การต่อขนาน

ในวงจรขนาน แต่ละแขนงต่ออยู่ระหว่าง node คู่เดียวกัน

ดังนั้น:

- แต่ละแขนงเห็นแรงดันเท่ากัน
- กระแสแยกไปตามแต่ละแขนง
- กระแสรวมคือผลรวมของกระแสแขนง
- การเพิ่มแขนงใหม่มักทำให้ความต้านทานเทียบเท่าลดลง
- ถ้าแขนงหนึ่งเปิดวงจร แขนงอื่นอาจยังทำงานต่อได้

สำหรับตัวต้านทาน:

`1 / R_total = 1 / R1 + 1 / R2 + 1 / R3 + ...`

สำหรับตัวต้านทานสองตัว:

`R_total = (R1 × R2) / (R1 + R2)`

ตัวอย่าง:

`R_total = (100 × 220) / (100 + 220) = 68.75 Ω`

ถ้าใช้แหล่งจ่าย `12 V`:

- กระแสในแขนง `100 Ω`: `12 / 100 = 0.12 A`
- กระแสในแขนง `220 Ω`: `12 / 220 = 0.0545 A`
- กระแสรวม: ประมาณ `0.1745 A`

วงจรขนานดึงกระแสมากกว่าการเอาตัวต้านทานชุดเดียวกันไปต่ออนุกรมมาก

</section>

<section>

## 5. กฎกระแสและแรงดันแบบมือใหม่

จำแบบสั้นได้แบบนี้:

| วิธีต่อ | สิ่งที่ใช้ร่วมกัน | สิ่งที่ถูกแบ่ง |
| --- | --- | --- |
| อนุกรม | กระแส | แรงดัน |
| ขนาน | แรงดัน | กระแส |

ตารางนี้ไม่ใช่แค่ท่องจำ

มันมาจากแนวคิดอนุรักษ์ที่ลึกกว่า:

- ประจุไม่สามารถกองสะสมที่ node ปกติได้เรื่อย ๆ
- พลังงานที่ได้จากแหล่งจ่ายต้องถูกถ่ายโอนไปตาม loop

แนวคิดที่ลึกกว่านี้จะกลายเป็น Kirchhoff's Current Law และ Kirchhoff's Voltage Law ในบทต่อไป

</section>

<section>

## 6. ทำไมความต้านทานอนุกรมจึงบวกกัน

ความต้านทานคือการขัดขวางในเส้นทาง

ถ้ากระแสต้องผ่านตัวต้านทานหนึ่ง แล้วผ่านอีกตัว แต่ละตัวจึงเพิ่มการขัดขวางให้เส้นทางกระแสเดียวกัน

ความต้านทานอนุกรมจึงเหมือนเพิ่มช่วงถนนแคบ ๆ เข้าไปบนถนนเส้นเดิม

ความต้านทานอนุกรมมากขึ้นหมายถึง:

- กระแสรวมน้อยลงเมื่อใช้แหล่งจ่ายแรงดันเท่าเดิม
- แรงดันรวมตกคร่อมเส้นทางมากขึ้น
- ตัวต้านทานแต่ละตัวได้รับส่วนแบ่งแรงดันจากแหล่งจ่าย

</section>

<section>

## 7. ทำไมความต้านทานขนานจึงเล็กลง

แขนงขนานสร้างเส้นทางให้กระแสเพิ่มขึ้น

ถ้ากระแสมีสองเส้นทางแทนที่จะมีเส้นเดียว วงจรรวมจึงนำกระแสได้มากขึ้นเมื่อแรงดันเท่าเดิม

ดังนั้นความต้านทานเทียบเท่าจึงต่ำลง

นี่คือเหตุผลที่ short circuit อันตราย: มันคือเส้นทางความต้านทานต่ำมากที่ต่อขนานกับโหลดที่ตั้งใจไว้

</section>

<section>

## 8. ตัวอย่างที่พบบ่อย

ตัวอย่างอนุกรม:

- สวิตช์ต่ออนุกรมกับหลอดไฟ
- ตัวต้านทานต่ออนุกรมกับ LED
- cell แบตเตอรี่ต่ออนุกรมเพื่อเพิ่มแรงดัน
- ฟิวส์หรือ breaker contact ต่ออนุกรมกับโหลดที่ต้องการป้องกัน

ตัวอย่างขนาน:

- โหลดหลายตัวในบ้านที่ต่ออยู่กับแหล่งจ่ายเดียวกัน
- pull-up หรือ pull-down path รอบ input ดิจิทัล
- cell แบตเตอรี่ต่อขนานเพื่อเพิ่มความจุ
- แขนงของวงจร sensor divider
- bypass capacitor ต่อคร่อมรางไฟเลี้ยง

วงจรจริงจำนวนมากผสมทั้งอนุกรมและขนาน

เครื่องชาร์จโทรศัพท์ แหล่งจ่ายไฟ แอมป์ หรือบอร์ด embedded ไม่ได้เป็น “อนุกรมล้วน” หรือ “ขนานล้วน” แต่มันคือ network ของทั้งสองแบบ

</section>

<section>

## 9. Open circuit และ short circuit

**Open circuit** หมายถึงเส้นทางถูกตัดขาด

ในวงจรอนุกรม ชิ้นส่วนหนึ่งที่เปิดวงจรอาจทำให้ทั้งเส้นทางหยุดทำงาน

**Short circuit** หมายถึงเส้นทางความต้านทานต่ำที่ไม่ตั้งใจให้เกิด

ในวงจรขนาน แขนง short อาจดึงกระแสส่วนใหญ่ไป เพราะกระแสไหลผ่านเส้นทางที่ง่ายกว่าได้มากกว่า

<aside className="callout">

**หมายเหตุความปลอดภัย:** “ความต้านทานต่ำ” ไม่ได้แปลว่าดีเสมอไป ความต้านทานต่ำคร่อมแหล่งจ่ายที่จ่ายกระแสได้มาก อาจทำให้กระแสสูง สายร้อน อาร์ก แผลไหม้ ไฟไหม้ แบตเตอรี่เสีย หรืออุปกรณ์ป้องกันตัดวงจร

</aside>

## 10. นิสัยการวัดที่ควรรู้

การวัดแรงดัน ให้วัดคร่อมสองจุด

นี่คือการวัดแบบขนาน

การวัดกระแส มิเตอร์ต้องเข้าไปเป็นส่วนหนึ่งของเส้นทางกระแส

นี่คือการวัดแบบอนุกรม

นี่คือเหตุผลที่การเอา ammeter ไปต่อคร่อมแบตเตอรี่หรือแหล่งจ่ายโดยตรงอาจอันตราย เพราะมิเตอร์อาจกลายเป็นเส้นทางขนานความต้านทานต่ำ

## 11. ความเข้าใจผิดที่พบบ่อย

- **“กระแสถูกใช้หมดในตัวต้านทานตัวแรก”** ในเส้นทางอนุกรมอย่างง่าย กระแสที่เข้าและออกจากแต่ละชิ้นส่วนเท่ากัน
- **“ขนานปลอดภัยกว่าเสมอ”** วงจรขนานอาจดึงกระแสรวมจากแหล่งจ่ายมากขึ้น
- **“รูปร่างของรูปวาดเป็นตัวบอกว่าอนุกรมหรือขนาน”** การต่อ node ต่างหากที่เป็นตัวบอก
- **“ชิ้นส่วนสองตัวแตะจุดเดียวกันคือขนาน”** ต้องใช้ node ร่วมกันทั้งสองด้านจึงจะเป็นขนาน
- **“ตัวต้านทานค่ามากกว่าจะได้กระแสมากกว่าเสมอ”** ในวงจรขนาน ตัวต้านทานค่าต่ำกว่าจะได้กระแสมากกว่า
- **“ทุกแขนงมีผลต่อกันเท่ากันหมด”** แหล่งจ่ายจริง สายไฟ tolerance ความร้อน และขีดจำกัดของอุปกรณ์ป้องกันมีผลเสมอ

</section>

## 12. ความปลอดภัยและบริบทประเทศไทย

แนวคิดอนุกรมและขนานปรากฏในอิเล็กทรอนิกส์ขนาดเล็ก แบตเตอรี่ ระบบไฟอาคาร ระบบป้องกัน solar array, EV pack และตู้ควบคุมอุตสาหกรรม

บทนี้ปลอดภัยในฐานะแบบจำลองแนวคิดแรงดันต่ำ

อย่าใช้บทนี้เพียงอย่างเดียวในการออกแบบงานไฟฟ้าอาคาร การประสานอุปกรณ์ป้องกัน battery pack, PV string, ระบบ EV หรือการติดตั้งจริง

สำหรับงานปฏิบัติในประเทศไทย บทความด้านความปลอดภัยในอนาคตต้องตรวจสอบข้อกำหนดล่าสุดจากแหล่งทางการไทยและสากล เช่น มอก./TIS, วสท., กฟน., กฟภ., กกพ., หน่วยงานท้องถิ่น เอกสารผลิตภัณฑ์ และมาตรฐาน IEC/IEEE ที่เกี่ยวข้อง

</section>

## เส้นทางเชิงเทคนิค

ตัวต้านทานอนุกรม:

`R_eq = ΣR_i`

`I_total = V_source / R_eq`

`V_i = I_total × R_i`

ตัวต้านทานขนาน:

`G_eq = ΣG_i = Σ(1 / R_i)`

`R_eq = 1 / G_eq`

`I_i = V_source / R_i`

`I_total = ΣI_i`

สำหรับตัวต้านทานเชิงเส้น กำลังในแต่ละส่วนคือ:

`P_i = V_i × I_i = I_i²R_i = V_i² / R_i`

ถ้ามองเป็นกฎวงจรทั่วไป:

- Kirchhoff's Current Law: ผลรวมกระแสที่ node เท่ากับศูนย์
- Kirchhoff's Voltage Law: ผลรวมการเปลี่ยนแปลงแรงดันรอบ loop เท่ากับศูนย์

กฎเหล่านี้มักเป็นขั้นจริงจังถัดไปหลังจากเข้าใจวงจรอนุกรมและขนานแบบพื้นฐาน

</section>
